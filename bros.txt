loading instruction

Turn on the floppy disk drive and turn off your Atari XL/XE. Now insert the Tobot disk face down into drive one and power on the XL/XE while holding the OPTION button. The program is now loaded automatically.

game description

The task is to first collect all the keys in each of the 50 rooms of the castle and then leave the room through a door. A lot of dangers like Deadly spikes, boulders, crumbling floors, holes, and spiders (which can only wound Tobot's legs) await Tobot. Tobot can use the laser to clear boulders. He can also build laser bridges that disappear when you walk over them. Keys can only be collected when Tobot stands directly below them and jumps up. Some floors then can be blasted. In addition, Tobot has the ability to teleport. A model will then be left where Tobot stood. The model should never be shot down! Just like that, no keys should eat! Spin a Each shot and each laser bridge costs Tobot energy. If Tobot has no more energy, he can either shoot or build bridges. This can be increased again on energy chargers.

steering

Control is via joystick in port one. You can run left and right, down, climb ladders, jump or blast down diagonally. With the fire button held down, To bot can shoot left and right, diagonally down/right down to build the laser bridge below, charge for energy (if he's in front of a charger) and jump up to grab a key. Tobot is teleported using the keyboard (space key). The Tobot model is brought to where Tobot is currently standing by pressing the "S" key.
